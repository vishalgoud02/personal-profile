Task

Build a app for language training services.

  • Strategy UX Strategy
  • Design UI/UX Design, Mobile App design
  • Feature
    • Native Speakers
    • Flexible schedule
    • Live Session

Story Behind

The client had to create a learner and trainer oriented interface where they could create their profiles, upload videos and also do live sessions.

Task

Solutions

We did a little user research to see how we could create an interface that would help both the learner and the translator and not cause any disturbance in the live session.
