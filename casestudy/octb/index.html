Task

Build a website for language training services.

  • Strategy UX Strategy

Story Behind

Acquiring a new language necessitates resources that engage individuals in ongoing, tailored instruction from educators.

Solution

I created a solution that not only optimizes productivity and effectiveness but also provides a user-friendly experience that enhances overall satisfaction and enjoyment.

Scheduling Lessons

Pupils have the option to book a class with their preferred instructor. A personalized electronic wallet manages all student payments and teacher compensation.

Instant Messaging

Design chat system for a instant messaging system with using the API.

Dashboard

Upon logging in, users are directed to their own customized dashboard where they can conveniently track their errors and advancements from their lessons.

Progress Reports

Created different view screens for different reports so that the report can be understood easily.
